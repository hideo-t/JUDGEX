# デモ・サンプル入力例

PRE-JUDGE Suiteをテストするためのサンプル入力集

## Layer 1（未整理）になる例

### ケース1: 情報不足
**Q1: 迷いは何？**
```
仕事を変えたい
```

**Q2: 選ばなければならないことは何？**
```
今のままか違う道か
```

**Q3: 決めたら引き受けるものは何？**
```
わからない
```

→ **期待される結果**: Layer 1、追加の深掘り質問が提示される

---

## Layer 2（判断以前）になる例

### ケース2: 決定対象は明確だが引き受けるものが不明
**Q1: 迷いは何？**
```
営業職から企画職に転職するか迷っている。
現在30歳で、今の会社で5年働いている。
```

**Q2: 選ばなければならないことは何？**
```
営業を続けるか、未経験の企画職に転職するか。
転職するなら今年中に決めたい。
```

**Q3: 決めたら引き受けるものは何？**
```
転職すると年収が下がるかもしれない。
営業を続けると新しいスキルが身につかない。
```

→ **期待される結果**: Layer 2、引き受けの言語化が促される

---

## Layer 3（判断可能）になる例

### ケース3: 完全に整理された状態
**Q1: 迷いは何？**
```
大阪の親会社に転勤するか、東京で転職するか迷っている。
家族（妻と子供2人）がいて、子供の学校のことも考える必要がある。
決断期限は3ヶ月後の4月末。
```

**Q2: 選ばなければならないことは何？**
```
①大阪転勤を受ける（昇進、年収100万円アップ）
②東京で転職する（年収は横ばい、未経験の業界）
③今の部署に残る交渉をする（昇進は見送り）
```

**Q3: 決めたら引き受けるものは何？**
```
①を選ぶと、家族の生活環境が変わり、子供の転校、妻の仕事探しが発生する。
②を選ぶと、年収アップを諦め、新しい環境でのリスクを取る。
③を選ぶと、キャリアの停滞と後悔の可能性がある。
いずれにしても、何かを失う覚悟が必要。
```

→ **期待される結果**: Layer 3、JUDGEX²計測が可能になる

---

## JUDGEX²のテストパターン

### パターンA: 引き受け傾向（高スコア）
- **J1**: A - 自分が引き受けて修正する
- **J2**: A - 不完全でも決めて動く
- **J3**: A - 最初に明文化してから決める

**期待スコア**: 100点、ラベル「引き受け傾向」

---

### パターンB: 保留傾向（中スコア）
- **J1**: B - 半分は状況、半分は自分
- **J2**: B - 追加情報を集めてから決める
- **J3**: B - 決めた後に調整しながら背負う

**期待スコア**: 64点、ラベル「保留傾向」

---

### パターンC: 回避傾向（低スコア）
- **J1**: D - 決定そのものをなかったことにしたくなる
- **J2**: D - 決めない理由を固める
- **J3**: D - 見ないようにして決定を避ける

**期待スコア**: 0点、ラベル「回避傾向」

---

## テストシナリオ

### シナリオ1: Layer 1 → Layer 3への遷移
1. Layer 1になる入力をする
2. 深掘り質問に具体的に答える
3. Layer 2または3に進む
4. 必要に応じて引き受けを言語化
5. Layer 3でJUDGEX²を計測

### シナリオ2: 最初からLayer 3
1. 完全に整理された入力（ケース3）を使用
2. すぐにJUDGEX²が利用可能になることを確認
3. 各パターンでスコアが正しく計算されることを確認

---

## デバッグ用の確認ポイント

### 入力時
- [ ] 文字カウンターが正しく動作するか
- [ ] 10文字未満で警告が表示されるか

### MAP生成時
- [ ] Layer判定が正しいか
- [ ] 詰まり点が最大3つ表示されるか
- [ ] 判断可能条件（問い）が最大3つ表示されるか
- [ ] Layer別のボタンが正しく表示されるか

### JUDGEX²
- [ ] Layer 3でないとボタンが表示されないか
- [ ] 3つの質問すべてに答えるまでボタンが無効か
- [ ] スコアが正しく計算されるか（0-100）
- [ ] 傾向ラベルが正しく表示されるか
- [ ] レーダーチャートが描画されるか

### UI/UX
- [ ] アニメーションがスムーズか
- [ ] レスポンシブデザインが機能するか
- [ ] スクロールが自動で上に戻るか
- [ ] リセットボタンで初期状態に戻るか
